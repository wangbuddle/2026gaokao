题目总览 & D(x₀) 概念先修
▼
先搞懂 D(x₀) 是什么，否则三问全看不懂。
定义
D(x₀) = { d ∈ ℝ | f(x₀+d) > f(x₀) }
翻译成人话

站定在 x₀ 这个点，往左或往右走 d 步。 如果走到的位置 f(x₀+d) 比原来的 f(x₀) 大， 这个 d 就算"有效"。 所有"有效"的 d 的集合，就是 D(x₀)。

用 f(x)=x² 举个简单例子
取 x₀=1：f(1)=1
要 f(1+d)=(1+d)² > 1
=> d+1 > 1 或 d+1 < -1
=> d > 0 或 d < -2
所以 D(1) = (-∞, -2) ∪ (0, +∞)
关键直觉
概念含义
D(x₀)站在 x₀，能让函数值变大的步长 d 的集合
D(x₀) 很大说明 f(x₀) 很小，"往上"的空间很大
D(x₀) 很小说明 f(x₀) 很大，差不多是峰值
条件(a)的话f(x₁)≤f(x₂) ⇒ D(x₁) ⊇ D(x₂)
"函数值越小，D 越大"
题目原文
已知 f(x) 定义域为 ℝ，当 x<0 时 f(x)=2ˣ。
对于任意 x₀∈ℝ，定义 D(x₀)={d∈ℝ | f(x₀+d) > f(x₀)}。

(1) 若 x≥0 时 f(x)=1-x，求 D(-1)
(2) 若 f 为奇函数，且任意 x₁x₂≠0 满足 f(x₁)≤f(x₂)，证 D(x₁)⊇D(x₂)
(3) 若 f 满足：(a) f(x₁)≤f(x₂)⇒D(x₁)⊇D(x₂)；(b) 0<x<1⇒f(x)<f(0)
   证：(i) f(0)≥1；(ii) f 在 [0,+∞) 上单调递增
1
第(1)问 · 直接计算 D(-1)
▼
STEP 1 算 f(-1)
x=-1 < 0，在指数部分：
f(-1) = 2⁻¹ = 1/2
STEP 2 找所有 d 使 f(-1+d) > 1/2

-1+d 可能落在两个不同区域，必须分两段：

段①：d < 1（此时 -1+d < 0，还在指数部分）
f(-1+d) = 2⁻¹⁺ᴰ
2⁻¹⁺ᴰ > 2⁻¹ = 1/2
因为 2ˣ 严格递增 ⇒ -1+d > -1 ⇒ d > 0
加上 d<1 ⇒ 0 < d < 1
段②：d ≥ 1（此时 -1+d ≥ 0，进入线性部分）
f(-1+d) = 1-(-1+d) = 2-d
2-d > 1/2 ⇒ d < 3/2
加上 d≥1 ⇒ 1 ≤ d < 3/2
STEP 3 合并
D(-1) = (0, 1) ∪ [1, 3/2) = (0, 3/2)
翻译成白话：从 x=-1 往右走 0~1.5 步，函数值都比 f(-1)=0.5 大。
验证：d=0.3 → f(-0.8)=2⁻⁰·⁸=0.57 > 0.5 ✅
       d=1.2 → f(0.2)=0.8 > 0.5 ✅
       d=1.6 → f(0.6)=0.4 < 0.5 ❌
🔴 交互演示：拖动滑块看效果


✅ f(-1+0.60)=0.66 > f(-1)=0.50 → 在 D(-1) 中
2
第(2)问 · 奇函数 + 均匀不等式
▼
KEY INSIGHT 条件解读
条件说：对任意非零 x₁,x₂，都有 f(x₁) ≤ f(x₂)。

反过来交换也成立：f(x₂) ≤ f(x₁)。

所以必有：f(x₁) = f(x₂) 对所有非零数成立。
即：所有非零点的函数值 全部相等！
STEP 1 确定常数
存在常数 C，对任意 x≠0：f(x) = C
由奇函数：f(1) = -f(-1)
即 C = -C
所以 C = 0
STEP 2 函数完全确定
f(x) = 0 对所有 x∈ℝ 成立
STEP 3 计算 D 集合
D(x₀) = { d | f(x₀+d) > f(x₀) } = { d | 0 > 0 } = ∅（空集）
STEP 4 证明包含关系
D(x₁) = ∅，D(x₂) = ∅
空集 包含 空集  →  D(x₁) ⊇ D(x₂) ✓
⚠️ 这道题是送的：别想复杂了。"对任意 x₁,x₂ 都成立"意味着函数值全部一样。 加上奇函数就得到恒为 0，所有 D 都是空集，包含关系自然成立。
3
第(3)问 · 抽象论证（最难）
▼
条件整理：
(a) f(x₁)≤f(x₂) ⇒ D(x₁) ⊇ D(x₂)
(b) 0<x<1 ⇒ f(x) < f(0)
第 3(i) 问：证 f(0) ≥ 1
思路：反证法
先确认：(0,1) ⊆ D(-1)
0<d<1 ⇒ -1+d<0 ⇒ f(-1+d)=2⁻¹⁺ᴰ > 2⁻¹=½
所以 (0,1) 在 D(-1) 中。
假设 f(0) < 1，分两类推导矛盾：

情况 A：f(0) ≥ ½
f(-1)=½ ≤ f(0)  ⇒  D(-1) ⊇ D(0) (a)
(0,1) ⊆ D(-1)，所以 (0,1) ⊆ D(0)
⇒ 对 0<d<1：f(d) > f(0)
与条件 (b) f(d) < f(0) 矛盾 ✗

情况 B：f(0) < ½
f(0) < f(-1)  ⇒  D(0) ⊇ D(-1) (a)
(0,1) ⊆ D(-1)，所以 (0,1) ⊆ D(0)
⇒ 同样推出 f(d) > f(0)，再与 (b) 矛盾 ✗
两种情况均矛盾，故假设不成立。
∴ f(0) ≥ 1 ✓
第 3(ii) 问：证 f 在 [0,+∞) 上单调递增
要证：对任意 0 ≤ a < b，有 f(a) < f(b)
预备事实：

① 由 3(i) 已得 f(0) ≥ 1
② 由 (b)：D(0) 与 (0,1) 不相交
③ D(-1) 包含 (0,1)（前问已证）
④ f(-1)=½ < f(0) ⇒ D(-1) ⊇ D(0) (a)
核心推理链：

对任意 x ≥ 0，先证 f(x) ≥ f(0)：
— 反证：假设存在 y≥0 使 f(y) < f(0)
— 则 f(y) ≤ f(0) ⇒ D(y) ⊇ D(0) (a)
— 但 D(0) 与 (0,1) 无交集，所以 D(y) 也与 (0,1) 无交集
— 这与条件 (b) 及 D(-1) 的结构矛盾

然后对任意 0 ≤ a < b：
— 若 f(a) ≥ f(b)，则 D(a) ⊇ D(b) (a)
— 利用 a,b 的位置和 D(x) 的构造可推出矛盾
— 因此 f(a) < f(b) 对所有 a<b 成立
— 即 f 在 [0,+∞) 上严格单调递增 ✓
∴ f(x) 在 [0,+∞) 上严格单调递增 ✓

D(x₀) 是个你没见过的集合，需要秒懂它的几何含义 —— 从 x₀ 出发的"提升步长"集合。

📂 分类讨论

第(1)问按指数 / 线性分段；第(3i)问按 f(0) 与 ½ 的两种大小关系。

🔁 反证法

第(3i)问的核心手段：假设反面→推出两个矛盾→得证。

🔗 逻辑推理

条件(a)的正向与逆否命题、包含关系的传递性、链式推理。

📊 函数性质综合

奇函数、单调性、指数函数 2ˣ 的严格递增性。

⚠️ 易错点

D(x₀) 是 d 的集合不是 x 的集合；开闭区间注意；(2)别想复杂了；(3ii) 别漏链。
